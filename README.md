This is a Backend for Training Sessions 

Technologies :
* Node.js
* Express
* MongoDB Atlas
  
Features :
* RESTful API built with Node.js and Express.js
* Modular architecture (routes, controllers, models, middlewares)
* JWT authentication with access and refresh tokens
* Role-based authorization
* Password hashing using bcrypt
* MongoDB integration with Mongoose ODM
* Data relationships and population
* AI-powered statistics generation using Google Gemini API
* Rate limiting to prevent abuse and brute-force attacks
* CORS configuration for secure frontend-backend communication
* HTTP security headers using Helmet
* Environment variable management with dotenv
* API testing using Postman
* Helmet Middleware for secure Http Headers (preventing XSS,Clickjacking...)

  
This is a basic beginner project. Going Forward..
